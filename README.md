# Flap Clash app privacy policy

Flap Clash uses Firebase authentication for user login. The preferred sign in is Google Play Games or Apple Game Center. When signing in with those, the app developers have no identifiable information. When signing in with email and password, they are secured by Firebase and the developers can see emails. To delete an email from the record, request so [here](https://docs.google.com/forms/d/e/1FAIpQLScie3D68i-K7OBtHljVZgwC8AKVFLaN7esgxtBuQahW5-Sy9w/viewform?usp=dialog).

All other data collected for Flap Clash is related to the game play and nothing else. It is tied to an anonymous user ID.

Flap Clash uses Google Crashlytics to help developers fix bugs, the data captured is anonymous.
